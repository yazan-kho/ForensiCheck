💡 What does Entropy mean?

Entropy measures how chaotic or random the data inside a file is, on a scale from 0 to 8.

  • Low Entropy (0-4): The data is highly structured or repetitive (e.g., plain text, empty files).
  • Medium Entropy (4-7): Normal structure with some complexity (e.g., most uncompressed images, audio, documents).
  • High Entropy (7-8): Highly random data. This usually means the file is compressed (like a ZIP archive) or encrypted.

By comparing the expected randomness of a file format to its actual randomness, ForensiCheck can spot files that might be hiding malicious data or claiming to be a different format than they actually are.
